<p>Welcome to the Banner Editor! This package allows you to upload multiple banner images or simply add text to create a banner without an image.</p>
<hr />
<p><strong>A couple pro tips:</strong></p>
<p>There's configurations on the property so make sure you set them after you install!</p>
<p>You can find an example front-end file in <code>/Views/Partials/OrcBannerEditorExample.cshtml</code>. Feel free to rename/reuse/edit it for your implementation!</p>
